# 📄 Resume Analyzer

A modern AI-powered Resume Analyzer built with Python and Streamlit.

It extracts resume text, identifies skills, calculates ATS score, and gives improvement suggestions.

---

## 🚀 Features

- 📂 Upload Resume (PDF)
- 🧠 Extract Text from Resume
- 🛠 Skill Detection
- 📊 ATS Score Calculation
- 📈 Resume Quality Rating
- 💡 Smart Suggestions
- 📄 Resume Preview
- 📥 Download Report
